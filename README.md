# Bookshop-Management-System

## Everyday is a learning day !

## Overview
The Bookshop Management System helps you manage books, customers, and orders in a digital bookstore environment. It allows admin to add and manage books, and customers to register, browse, and purchase books. This project uses .NET (C#) for the backend, React.js for the frontend, and MySQL as the database to provide a scalable and maintainable system.

Features

- Admin can register and login to manage the bookstore.
- Customers can register, login, and purchase books.
- Books can be added, listed, and purchased with proper tracking.
- Supports secure authentication for both admin and customers.

## Services
- Admin Service
- Customer Service
- Book Service
- Order Service

## Prerequisites
- .NET
- MySQL Server
- Visual Studio Code
